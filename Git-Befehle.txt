Git-Befehle :

Änderung auf eigenen Branch hochladen 
1. git status
2. git add (DAteiname)
3. git commit -m "NAchricht"
4. git push 

voraussgesetzt oben schon gemacht 
Änderung auf main branch hochladen :
1. git checkout main 
2. git merge (eigener branch  name )
3. git push 


vom main herunterladen auf eigenen Branch  :
1. git checkout main 
2. git pull 
3. git checkout (eigener branch name ) 
4. git merge main 
5. git push
